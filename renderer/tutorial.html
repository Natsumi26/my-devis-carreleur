Introduction

Bienvenue dans Mes DevisFactures ! Ce tutoriel vous guide pas à pas dans l'utilisation des principales fonctionnalités.

🎯 Objectif du logiciel

Ce logiciel vous permet de gérer vos devis et factures de manière simple et rapide, avec génération automatique en PDF, envoi par email, et suivi des événements.

🧭 Navigation générale

  • Accueil : Vue d'ensemble de vos devis et factures.
  • Clients : Liste des clients enregistrés dans une base de données.
  • Prestations : Liste des prestations associés à leurs prix unitaire enregistrés dans une base de données.
  • Créer un devis / facture : Formulaire intuitif pour saisir les informations client, prestations, et montants.
  • Planning : Visualisez vos rendez-vous et interventions.
  • Paramètres : Configurez vos informations d'entreprise, vos réglages SMTP pour l'envoi d'email et la gestion de la bdd.
  • Aide : Vous êtes ici 😊

👥 Gestion des clients

La section clients vous permet de visualiser tous les clients de votre base de données. Vous pouvez faire :

  • Rechercher par nom de client dans la barre de recherche
  • Trier sur la colonne code et nom du client
  • Ajouter un nouveau client comme ci-dessous
  • Exporter en excel la liste de vos clients
  • Importer en excel une liste de clients
  • Vous avez un menu contextuel avec le click droit sur une ligne de devis :
    1. Modifier : ouvre une modal qui permet de modifier les informations du client
    2. Supprimer : permet de supprimer le client

👥 Créer un client

Comment faire :

  1. Cliquez sur "Ajouter un client"
  2. Remplir les champs
  3. A la fin, vous cliquez sur "Ajouter" pour créer le client

📄 Gestion des devis

La section devis vous permet de visualiser tous les devis de votre base de données. Vous pouvez faire :

  • Rechercher par nom de client dans la barre de recherche
  • Trier sur la colonne date de devis
  • Ajouter un nouveau devis comme ci-dessous
  • Vous avez un menu contextuel avec le click droit sur une ligne de devis :
    1. Envoyer par mail : ouvre une modal avec un text par défaut modifiable pour envoyer le devis en pdf
    2. Voir le devis : ouvre une modal pour visualiser le pdf du devis avec possibilité de le télécharger ou de l'imprimer
    3. Ajouter au planning : permet d'ajouter le "chantier" à votre planning
    4. Accepter : permet de changer le statut du devis en "Accepté"
    5. Générer une facture : permet de créer une facture pour ce devis
    6. Télécharger : permet de télécharger le devis en pdf
    7. Modifier : ouvre une modal qui permet de modifier les informations du devis
    8. Supprimer : permet de supprimer le devis

📄 Créer un devis

La création d'un devis doit faire suite à la création ou l'import de clients et de prestations. Comment faire :

  1. Cliquez sur "Ajouter un devis"
  2. Les champs grisés ne sont pas modifiables
  3. Choisir un client
  4. Choisir un taux de TVA
  5. Le statut par defeut est "En attente" mais vous pouvez le modifier
  6. Ajouter les prestations, leurs quantités et l'unité de mesure
  7. Vous pouvez cliquer sur "Ajouter une prestation" pour en ajouter des supplémentaires
  8. Vous avez la corbeille pour supprimer la prestation
  9. Vérifiez le total et les taxes
  10. A la fin, vous cliquez sur "Ajouter" pour créer le devis

🧾 Gestion des factures

La section factures vous permet de visualiser toutes les factures de votre base de données. Vous pouvez faire :

  • Rechercher par nom de client dans la barre de recherche
  • Trier sur la colonne date de la facture
  • Ajouter une nouvelle facture comme ci-dessous

    2. Ajouter un acompte : ouvre une modal pour ajouter un acompte versé pour cette facture (date, montant, et mode de payement)
    3. Voir la facture : ouvre une modal pour visualiser le pdf de la facture avec possibilité de la télécharger ou de l'imprimer
    4. Télécharger : permet de télécharger la facture en pdf
    5. Supprimer : permet de supprimer la facture

📅 Gestion du planning/calendrier

Le planning est basé sur FullCalendar. Vous pouvez :

  • Vous connecter a votre Google Agenda vous visualisé votre agenda personnel.
  • Visualiser les événements des devis.
  • Créer un nouvel événement en cliquant sur une date (un évenement lié à une devis (professionnel) ou non (personnel)).
  • Modifier un événement en le sélectionnant.
  • Ajouter votre évenement à votre Google Agenda.
  • Supprimer un événement depuis la fenêtre de détails.

⚙️ Gestion des paramètres

Vous pouvez :

  • Passer en mode dark ou light.
  • Enregistrer les données de votre entreprise
  • Paramétrer l'envoi d'email
    • Type de boite mail (la votre pour les envois)
    • Port d'envoi (par défault 465)
    • Activer la sécurité (true ou false)
    • Votre adresse mail (si type de mail gmail alors une adresse mail gmail)
    • Mot de passe d'application (il faut le créer dans les paramètres de sécurité de votre boite mail, souvent sous le format : xxxx xxxx xxxx xxxx)
  • Réinitialiser la base de données (⚠️ Cela supprime toutes les données de votre base, si vous n'etes pas sur sauvegarder votre base de données avant).
  • Sauvegarde la base de données (fichier avec l'extension .sqlite qui contient toutes vos données).
  • Importer la base de données (fichier avec l'extension .sqlite d'une ancienne sauvegarde).

💡 Astuces

  • Vous pouvez utiliser la fonction de recherche pour retrouver rapidement un devis ou un client.
  • Il faut supprimer la facture avant le devis.
  • D'abord effectuer un export pour avoir le modèle excel et après modifier le fichier pour l'importer.
  • Lors de la desinstallation du logiciel, la base de données est supprimer sans posibilité de la récupérer.
